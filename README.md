## Ceedbox technical test

- `POST /shipments/:clientId`: Save a new shipment
- `GET /shipments`: Browse all shipments

### Getting Started

```
docker compose up --build
```

or

```
# local redis and mysql db needed before you run
nvm use
npm install
npm run dev
```

### Implemented Features

- Scalable Adapter Pattern
- Redis
- MySQL
- Startup DB Synchronization
- UI
- React using Server-Side Rendering (SSR)
- Security - Helmet & Rate Limiting
- Config
- Basic Authentication
- Docker
- Terraform
- Jest
- Logging - Winston
- Load Testing - Autocannon
- Linting - Prettier
- Git hooks - Husky
